Choices editor

The choices editor lists the configuration elements of the component. A choice provides a set of possible values for model parameters, parameters and other configurable elements.

Each choice has a mandatory name and at least one enumeration must be specified. Each enumeration has a mandatory value, an optional text to display instead of the value and an optional help description. When the value of e.g. a model parameter refers to the choice, the possible model parameter value is selected from the set of enumerations in that choice.
