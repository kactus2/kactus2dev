Parameters editor

Parameters editor can be used to add, remove and modify parameters of a component. Parameters are name-value pairs to configure or hold information related to the containing component.

Each parameter contains mandatory name and value, and optionally a freetext description. By selecting a choice, the user can restrict the allowed parameter values to a set of predefined values.
Note that these are not VHDL generics.

Parameters editor contains a context menu (right mouse button) that provides following options:

  • Add new parameter (Add row)
  • Remove parameter (Remove row)
  • Clear the selected cells
  • Copy the contents of the selected cells to clipboard
  • Paste the contents of the clipboard to the selected cells
  • Import a csv-file to the editor
  • Export the contents of the editor to a csv-file
